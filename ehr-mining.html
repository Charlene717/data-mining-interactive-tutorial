概覽

EHR 資料的雙重結構

電子病歷 (EHR) 同時包含結構化 (診斷、用藥、檢驗、生命徵象) 與非結構化 (出院摘要、影像報告、護理紀錄) 兩類資料。前者用標準術語 (ICD-9/10/11、CPT、LOINC、RxNorm、SNOMED CT) 編碼，後者需 NLP 萃取。任何一個臨床問題的探勘——再住院預測、敗血症早期警示、藥物副作用偵測——都必須先把這兩端整合成「分析就緒」資料表。

大型公開資料集 (MIMIC-IV 2023、eICU、All of Us、N3C) 與通用資料模型 (OMOP CDM 5.4、Sentinel CDM、PCORnet CDM) 讓跨機構分析變得可能；但 缺失非隨機 (MNAR)、編碼漂移 (code drift)、記錄者偏差 (charting bias) 是每個 EHR 探勘專案都得正面處理的三大陷阱。

EHR contains both structured (diagnoses, meds, labs, vitals) and unstructured (discharge summaries, imaging reports, nursing notes) data. The first uses standardized terminologies (ICD-9/10/11, CPT, LOINC, RxNorm, SNOMED CT); the second requires NLP extraction. Any clinical mining problem — readmission prediction, sepsis early warning, ADR surveillance — must first merge both sides into an analysis-ready table.

Large open resources (MIMIC-IV 2023, eICU, All of Us, N3C) and common data models (OMOP CDM 5.4, Sentinel CDM, PCORnet CDM) enable cross-institution analytics; but MNAR missingness, code drift, and charting bias are the three traps every EHR mining project must confront.

🚨
Charting bias：「沒有記錄 ≠ 沒有發生」。一個 ICU 病人沒有 GCS 紀錄，往往代表他清醒，而不是缺失。直接 mean-impute 會嚴重失真。 Charting bias: "Not charted ≠ did not happen." A missing GCS in an ICU patient often means alert, not missing. Naive mean-imputation will badly distort signal.

三、Computable Phenotype

表型 (phenotype) = 把「這個病人是不是 type 2 diabetes」的臨床定義轉成「在 EHR 上可以執行的查詢」。PheKB 與 eMERGE Network 提供經驗證、跨機構通用的演算法庫；OHDSI ATLAS 則讓你用 cohort definition JSON 跨網絡執行。

典型 T2DM 表型：(2 次門診 ICD-10 E11.* > 30 天) OR (1 次住院 E11.*) OR (HbA1c ≥ 6.5%) OR (處方降血糖藥) — 並排除 type 1 與妊娠糖尿病。

A phenotype turns the clinical definition of "this patient has type 2 diabetes" into an executable EHR query. PheKB and the eMERGE Network publish validated, cross-institution algorithm libraries; OHDSI ATLAS lets you run cohort-definition JSON across a network.

Canonical T2DM phenotype: (2 outpatient ICD-10 E11.* > 30 days apart) OR (1 inpatient E11.*) OR (HbA1c ≥ 6.5%) OR (antihyperglycemic prescription) — excluding type 1 and gestational diabetes.
