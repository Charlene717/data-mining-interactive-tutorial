CHAPTER 3 / 20 · MODULE A

探索性資料分析 (EDA) 與視覺化

在訓練任何模型之前，先讓資料說話——這是 Tukey 1977 年留給我們的最重要遺產。

Before training any model, let the data speak — Tukey's most important legacy from 1977.

概覽

EDA 的四個目的

探索性資料分析 (Exploratory Data Analysis, EDA) 由 John Tukey 在 1977 年提出，主張資料分析的第一步應該是「看」而不是「測」。它的四個目的：① 理解資料分佈與規模 ② 找出異常值與資料品質問題 ③ 形成假設與特徵發現 ④ 為後續建模選擇合適的模型族。

下圖是著名的 Anscombe's Quartet 範例：四組資料具有完全相同的平均值、變異數、相關係數與線性迴歸線，但分佈完全不同。光看統計量會被誤導——唯有畫圖才能揭露真相。

Exploratory Data Analysis (EDA), coined by John Tukey in 1977, argues the first step of analysis should be to look, not to test. Four goals: ① understand distribution and scale ② surface outliers and quality issues ③ generate hypotheses and feature ideas ④ pre-select model families for modeling.

The famous Anscombe's Quartet shows four datasets with identical mean, variance, correlation, and regression line — yet completely different shapes. Numbers alone mislead; only plots reveal truth.

圖型選擇

一、依資料類型選擇圖型

🔢 數值單變量

Histogram (整體分佈)、Boxplot (五數摘要 + outliers)、Violin (分佈密度 + 五數)、ECDF (累積分佈)、Q-Q plot (是否近似常態)。

Histogram (overall shape), Boxplot (5-number + outliers), Violin (density + 5-number), ECDF (cumulative), Q-Q plot (normality check).

🏷️ 類別單變量

Bar chart、Lollipop、Pareto chart。避免 3D 圓餅圖——人眼難以比較弧長。

Bar chart, Lollipop, Pareto chart. Avoid 3D pie charts — humans cannot compare arc lengths reliably.

🔗 雙變量（數值 × 數值）

Scatter (大量點時用 hexbin / 2D density)、LOESS 平滑曲線、聯合分佈圖。

Scatter (use hexbin / 2D density for many points), LOESS smoother, joint distribution plot.

🎭 雙變量（類別 × 數值）

Grouped boxplot / violin、Strip + jitter、Ridgeline plot；對小樣本顯示原始點。

Grouped boxplot/violin, strip + jitter, ridgeline plot; always show raw points for small N.

🌐 多變量

Pairplot (scatter matrix)、Correlation heatmap (色階 viridis)、Parallel coordinates、PCA biplot、UMAP/t-SNE。

Pairplot (scatter matrix), correlation heatmap (use viridis), parallel coordinates, PCA biplot, UMAP/t-SNE.

⏱️ 時序

Line + trend、Calendar heatmap、Seasonal decomposition (STL)、Spaghetti plot (多個體)。

Line + trend, calendar heatmap, seasonal decomposition (STL), spaghetti plot (multi-subject).

決策樹

二、視覺化設計的 7 大原則 (Tufte / Wilkinson)

🌳 圖表選擇決策樹

Q1:
你要呈現「比較」？ → 是 → 橫向 bar chart（避免 3D 圓餅）。
Q2:
你要呈現「分佈」？ → 是 → 小樣本 strip + violin；大樣本 histogram / density。
Q3:
你要呈現「關係」？ → 是 → 數值-數值用 scatter；分類-數值用 grouped box；分類-分類用 mosaic / heatmap。
Q4:
你要呈現「組成」？ → 是 → Stacked bar、treemap、sunburst；避免 3D。
Q5:
你要呈現「演化」？ → 是 → 時序：line + ribbon (CI)；多個體：spaghetti + median bold。
Q1:
Show "comparison"? → Yes → Horizontal bar chart (avoid 3D pies).
Q2:
Show "distribution"? → Yes → Small N: strip + violin; Large N: histogram / density.
Q3:
Show "relationship"? → Yes → Num-num: scatter; cat-num: grouped box; cat-cat: mosaic / heatmap.
Q4:
Show "composition"? → Yes → Stacked bar, treemap, sunburst; avoid 3D.
Q5:
Show "evolution"? → Yes → Time: line + CI ribbon; multi-subject: spaghetti + bold median.
⚠️
色彩規則：(1) 連續尺度用 viridis / cividis（色盲友善）；(2) 發散尺度用 RdBu；(3) 類別最多 7 色；(4) 紅綠對比避免（綠色色盲約 5%）。 Color rules: (1) Sequential — use viridis / cividis (CVD-safe); (2) Diverging — use RdBu; (3) Max 7 categorical colors; (4) Avoid red-green contrast (~5% red-green CVD).

實作：自動 EDA 報告

library(DataExplorer); library(ggplot2); library(GGally)
# 一鍵 EDA 報告 / One-shot EDA report
create_report(df, y = "target", output_file = "eda.html")

# 個別圖型 / Targeted plots
plot_missing(df)               # 缺失地圖 / missingness map
plot_histogram(df)             # 所有數值變數 histogram
plot_correlation(df, type="c") # 相關 heatmap (Spearman 較穩)

# 雙變量探索 / Bivariate exploration
ggpairs(df, columns = 1:6, aes(color = target))

# 自訂 ggplot 範例 — Tufte 風格 / Custom ggplot — Tufte style
ggplot(df, aes(age, glucose, color = sex)) +
  geom_point(alpha = .3) +
  geom_smooth(method = "loess") +
  scale_color_viridis_d() +
  theme_minimal(base_size = 13) +
  labs(title = "Age vs Glucose", x = "Age (yrs)", y = "Fasting glucose (mg/dL)")
import pandas as pd, seaborn as sns, matplotlib.pyplot as plt
from ydata_profiling import ProfileReport

# 一鍵 EDA 報告 / One-shot EDA report
ProfileReport(df, title="EDA", explorative=True).to_file("eda.html")

# 缺失矩陣 / Missingness matrix
import missingno as msno
msno.matrix(df); plt.show()

# Correlation heatmap (viridis)
sns.heatmap(df.corr(method="spearman"), cmap="viridis", center=0,
            annot=True, fmt=".2f"); plt.show()

# Pairplot (含類別著色) / Pairplot (hued)
sns.pairplot(df, hue="target", corner=True, palette="viridis"); plt.show()

# Tufte 風格 scatter
fig, ax = plt.subplots()
sns.scatterplot(data=df, x="age", y="glucose", hue="sex", alpha=.3, ax=ax)
